# Student-System
Student-System

Веб-приложение для управления студентами, их оценками и аналитикой. Построено на Spring Boot, Thymeleaf и PostgreSQL. Система работает как мини-админка для учебных заведений: преподаватели ведут журнал, студенты просматривают свою успеваемость, админы управляют всей системой.

Возможности
Для преподавателей

Просмотр списка студентов

Выставление оценок

Просмотр журнала конкретного студента

Раздел «Аналитика»: графики, распределение оценок и статистика

Для студентов

Личный кабинет

Просмотр успеваемости по предметам

История оценок

Для администратора

Управление пользователями

Просмотр и редактирование данных

Панель администратора с дашбордом
