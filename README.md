# Student-System
Веб-приложение для управления данными студентов, оценками и аналитикой. Построено на Spring Boot, Thymeleaf и PostgreSQL, работает в формате микро-админки для школ/универов. Проект позволяет преподавателям вести учёт успеваемости, студентам — смотреть свои журналы, а админам — контролировать общую систему.

Функционал
Для преподавателей

Просмотр списка студентов

Выставление оценок

Просмотр индивидуального журнала студента

Раздел «Аналитика»: визуализация распределения оценок и статистики

Для студентов

Личный кабинет

Просмотр успеваемости по предметам

История оценок

Для администратора

Панель управления

Управление студентами и преподавателями

Дашборд с графиками (Chart.js)

Архитектура

Приложение построено на классической MVC-структуре:

Controller — обработка запросов

Service — бизнес-логика

Repository — работа с базой

Entity — модели данных

Оценки и статистика собираются через сервисы, которые формируют данные для диаграмм на фронте.
